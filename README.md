# Apresentação

## Bibliotecas utilizadas:
      
shadcn/ui - componentes já prontos

phosphor - icones

## Comandos

* Inicializar o shadcn:
````
      npx shadcn@latest init
````

* Instala o componente de botão do shadcn:
````
      npx shadcn@latest add button
````

* Instala o phosphor:
````
      npm install @phosphor-icons/react
````
